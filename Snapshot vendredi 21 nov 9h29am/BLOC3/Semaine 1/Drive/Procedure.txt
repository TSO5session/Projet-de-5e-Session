1.	
Téléchargez la denière version de FCT.
2.	
Une fois le logiciel installé, sélectionnez dans un nouveau projet le contrôleur CMMO-ST avec le vérin EPCO adéquat.
3.	
Suivez les neuf étapes de paramétrage du contrôleur.
 	3.1.	
Sélectionnez le mode de contrôle “binary profile 31”.
 	3.2.	
Définissez les fonctions du contrôleur qui vous seront utiles.
 	3.3.	
Fixez les valeurs limites telles que la vitesse et l'accélération.
 	3.4.	
Précisez la méthode de prise de référence.
 	3.5.	
Ajustez les données pré-calculées du contrôleur pour la combinaison EPCO.
 	3.6.	
Réglez les options de déplacement au ralenti. 
 	3.7.	
Remplissez la table de positionnement et précisez les paramètres tels que la position, la vitesse, la poussée et le jerk limite.
 	3.8.	
Définissez les messages : déplacement terminé, déviation du contrôleur, contrôle d'arrêt.
 	3.9.	
Définissez la gestion des erreurs : le report doit-il avoir lieu, si oui, comment ?
4.	
Après avoir complété le projet, connectez le CMMO-ST à votre ordinateur à l'aide d'un câble réseau.
5.	
L'adresse IP du contrôleur est automatiquement enregistrée dans le projet.
6.	
Téléchargez l'ensemble du projet depuis votre ordinateur dans le contrôleur.
+ Opérations de contrôle à distance depuis un PC sans intermédiaire d'un API
